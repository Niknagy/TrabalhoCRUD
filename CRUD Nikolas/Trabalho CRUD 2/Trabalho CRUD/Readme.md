# Sistema de Controle de Tarefas - CRUD Python & SQLite

## 🎯 Objetivo do Projeto

Este projeto é um sistema simples de **Controle de Tarefas** desenvolvido como proposta de trabalho de class. O objetivo principal é implementar um CRUD (Create, Read, Update, Delete) utilizando Python e persistência de dados local com SQLite. 

O sistema permite gerenciar tarefas de funcionários, acompanhando suas prioridades e status de conclusão.

## 💻 Tecnologias Utilizadas
* **Python 3:** Linguagem principal para a lógica de programação.
* **SQLite3:** Banco de dados relacional leve e nativo do Python para persistência dos dados.
* **VS Code / Terminal:** Ambiente de desenvolvimento e execução local (pode ser adaptado para rodar no GitHub Codespaces).

## 🚀 Instruções de Execução

Como o SQLite já é integrado ao Python, você não precisa instalar nenhum servidor de banco de dados extra. Para rodar o projeto na sua máquina local:

1. Certifique-se de ter o Python instalado.
2. Clone este repositório ou baixe os arquivos `main.py` e `database.py`.
3. Abra o terminal na pasta onde os arquivos estão salvos.
4. Execute o seguinte comando:

    -bash `python main.py`

## Nota: o app.db será criado automaticamente na primeira vez que o sistema for executado.
